## 🌟 Vision

**Spoonergy™** is not just a tool; it's a statement. Designed for those who demand perfection in every detail of their lives, Spoonergy™ combines advanced ergonomic research with aerospace-grade aesthetics. This landing page is a digital reflection of the product's soul: minimalist, high-fidelity, and technologically superior.

## ✨ Key Features

- **💠 3D Interactive Showcase**: Experience the Spoonergy™ in full 3D. Inspect every curve, highlight every detail, and feel the weight of design excellence.
- **🌊 Fluid Motion & Interactive Backgrounds**: Utilizing `Framer Motion` and custom WebGL shaders for a living, breathing background that reacts to your presence.
- **🛍️ Premium Checkout Flow**: A multi-stage, animated checkout experience that feels more like an onboarding to a luxury club than a simple purchase.
- **🌍 Global Intelligence (i18n)**: Native multilingual support in English and Spanish, ensuring the Spoonergy™ message resonates across borders.
- **⚡ Performance First**: Powered by **Astro 6**, achieving near-perfect Lighthouse scores while maintaining high-fidelity visuals.

## 🏗️ Technical Architecture

This project is built with a modern, high-performance stack:

- **Framework**: [Astro](https://astro.build/) (Static Site Generation with Selective Hydration)
- **UI Architecture**: [React](https://react.dev/) + [Tailwind CSS 4.0](https://tailwindcss.com/)
- **3D Engine**: [Three.js](https://threejs.org/) with [@react-three/fiber](https://r3f.docs.pmnd.rs/)
- **Motion Engine**: [Framer Motion](https://www.framer.com/motion/) & [Lenis](https://lenis.darkroom.engineering/) for smooth kinetic scrolling
- **State & Logic**: Custom hooks for interactive 3D inspection and checkout flow

## 🚀 Getting Started

To experience the development environment:

1. **Clone the repository**
   ```bash
   git clone https://github.com/nicolasreyes/spoonergy.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Ignite the server**
   ```bash
   npm run dev
   ```

4. **Visit** `http://localhost:4321`
